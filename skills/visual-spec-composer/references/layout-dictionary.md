# 布局骨架词典

本词典用于把常见构图 / 分块表达收敛成稳定骨架名，便于 VGS 填写与后续 adapter 转译。

## 单画面骨架

### 居中主体
- 主体居中
- 适合海报、KV、产品主视觉
- 通常配合留白或对称光线

### 左图右文
- 左图右文
- 适合需要标题区的横版 KV

### 右图左文
- 右图左文
- 与上相反

### 三分法
- 三分法视觉重心
- 适合摄影感、纪实感或电影感画面

### 对角动势
- 对角线动势
- 适合运动、动作、速度感场景

### 近景聚焦
- 特写主体
- 适合产品细节、人物情绪、材质展示

### 广角全景
- 远景建立镜头
- 适合世界观、场景、宏大空间

### 留白主体
- 主体 + 大留白
- 适合极简、标题区、品牌感场景

## 2. 多分块骨架

### 垂直分割
- 左右两块
- 适合 before/after、对照、双主题

### 水平分割
- 上下两块
- 适合上下叙事、信息层级分离

### 左主右辅
- 左大右小
- 适合主视觉 + 辅助说明

### 右主左辅
- 右大左小
- 与上相反

### 上二下三
- 上两块、下三块
- 适合 5 区拼贴、杂志感 montage

### 上三下二
- 上三块、下两块
- 信息重心偏上

### 中心四辅
- 中央主块 + 四周辅助块
- 适合一主多辅结构

### 二乘二网格
- 2x2 网格
- 稳定、清晰、易读

### 三乘二网格
- 3x2 网格
- 适合 6 区内容承载

### 条带序列
- 条带式连续叙事
- 适合时间流、镜头推进

### 编辑拼贴
- 不规则杂志拼贴
- 适合风格实验，但 one-shot 风险较高

### 辐射聚合
- 中心扩散式
- 适合中心主物 + 周边元素围绕

---

## 骨架选择建议

- 如果用户说“就一张图” → 优先单画面骨架
- 如果用户说“分成几块展示” → 优先多分块骨架
- 如果用户想要“杂志感 / collage / 不规则拼贴” → `editorial_collage`
- 如果用户既要主视觉又要几个小辅助块 → `center_hero_four_support` 或 hero/support 型骨架
- 如果模型较弱，优先：
  - `centered_hero`
  - `left_hero_right_support`
  - `split_vertical`
  - `grid_2x2`

---

## 4. 风险提示

以下骨架对小模型不友好：
- `top_2_bottom_3`
- `grid_3x2`
- `editorial_collage`
- `strip_sequence`（当每块内容差异很大时）

处理方式：
- 改成 two_stage
- 减少块数
- 先用 Human Brief 确认结构，再生成 Compact Prompt
